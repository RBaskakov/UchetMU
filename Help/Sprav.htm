Ведение справочной информации в программе Учет МедУслуг

Вызов справочников для редактирования осуществляется через пункт меню Справочники. Для работы программы необходимо заполнить следующие справочники:

Подразделения. Данный список содержит наименования подразделений, оказывающих платные медицинские услуги.  

Медицинские услуги.

Ведение данного справочника осуществляется с помощью двух таблиц. В верхней таблице отображается элементы справочника подразделений. При выборе подразделения в нижней таблице отображается список услуг, оказываемых выбранным подразделением. Каждая услуга характеризуется свойствами - наименование, единица измерения (выбирается из справочника единиц измерений), кодом услуги по номенклатуре и кодом услуги по подразделению

Единицы измерения услуг. Данный список содержит наименования единиц измерения медицинских услуг.

Виды расчета за услугу. Данный список содержит наименования видов расчета за услугу.

Виды медицинского страхования. Данный список содержит наименования видов медицинского страхования.

Страховые компании. Данный справочник содержит перечень страховых компаний, с указанием наименования, номера договора и даты договора со страховой компанией.

Медицинские полисы. Данный справочник содержит перечень медицинских полисов, с указанием фамилии, имени, отчества застрахованного лица, года его рождения, страховой компании, выдавшей медицинский полис (из справочника страховых компаний), серии и номера полиса.

Статьи калькуляции. Данный справочник содержит список статей калькуляции, с указанием на какую статью производится округления.

Заказчики. Данный список содержит наименования заказчиков медицинских услуг.
