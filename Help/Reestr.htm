Ведение реестров в программе Учет МедУслуг

Вызов реестров для редактирования осуществляется через пункт меню Реестры. В программе используются следующие реестры:

Реестр калькуляций.  

Ведение данного реестра осуществляется с помощью двух таблиц. В верхней таблице отображается список калькуляций. Для каждой калькуляции пользователь заполняет следующие реквизиты:

подразделение, оказывающее услугу (из справочника Подразделения)

после выбора подразделения становится доступным поле услуга. Значение поля услуга выбирается из справочника медицинских услуг для выбранного подразделения)

дата начала действия калькуляции

дата окончания действия калькуляции

В нижней таблице пользователь заполняет суммы статей калькуляции для выбранной в верхней секции калькуляции. Список статей калькуляции берется из справочника статей калькуляции.

Для добавления или удаления калькуляции щелкните правой кнопкой мышью в верхней секции и выберите соответствующий пункт контекстного меню. Если пользователь выбирает пункт меню Удалить, будет удалена выбранная калькуляция.

Реестр оказанных услуг.  

Реестр оказанных услуг может отображаться как полностью, так и в разрезе заказчиков и месяцев. В случае, если пользователей выберет отображение всего реестра оказанных услуг откроется форма для редактирования оказанных услуг

В верхней секции отображается список оказанных услуг. Для каждой услуги пользователь заполняет следующие реквизиты:

подразделение, оказывающее услугу (из справочника Подразделения)

после выбора подразделения становится доступным поле услуга. Значение поля услуга выбирается из справочника медицинских услуг для выбранного подразделения)

дата оказания услуги

вид страхования (из справочника видов медицинского страхования)

количество оказанной услуги

заказчик (из справочника заказчиков)

плательщик

После выбора услуги и даты ее оказания, в случае если на выбранную услугу существует калькуляция с периодом действия, в который попадает дата оказания данной услуги, то в поле "цена" подтягивается сумма из калькуляции. Если цена для выбранной услуги определена, после изменения количества оказанной услуги рассчитывается сумма оказанной услуги. В поле "оплаченная сумма"  отображается сумма всех оплат данной услуги, перечисленных в нижней секции.

Для добавления или удаления услуги щелкните правой кнопкой мышью в верхней секции и выберите соответствующий пункт контекстного меню. Если пользователь выбирает пункт меню Удалить, будет удалена выбранная услуга.

В нижней секции отображается список оплат для выбранной в верхней секции услуги. Для каждой оплаты пользователь заполняет следующие реквизиты:

вид оплаты (из справочника видов расчета за услугу)

номер счет фактуры

дата счет фактуры

номер платежного поручения

дата платежного поручения

назначение платежа

сумма оплаты

Для добавления оплаты заполните самую нижнюю строку таблицы. Для удаления оплаты выберите оплату и щелкните правой кнопкой мышью и выберите пункт контекстного меню Удалить. Если пользователь выбирает пункт меню Удалить, будет удалена выбранная калькуляция.

В случае, если пользователей выберет отображение реестра оказанных услуг по заказчикам и периодам, откроется форма для выбора заказчика и периода

В верхней таблице отображается список заказчиков с указанием сумм оказанных услуг (всего). В данной таблице отображаются только заказчики имеющие ненулевые суммы оказанных услуг. В нижней таблице для выбранного в верхней таблице заказчика отображаются суммы оказанных услуг в разрезе периодов. В данной таблице отображаются только периоды, в которых сумма оказанных услуг больше нуля. Для перехода к редактированию реестра оказанных услуг дважды щелкните мышью на выбранном периоде. Отобразится список услуг и их оплат для выбранного заказчика и периода. Откроется описанная выше формы оказанных услуг. Правила ее редактирования совпадает с правилами редактирования формы всех услуг, описанные выше, за исключением того, что поле заказчик заполняется автоматически и не отображается.
