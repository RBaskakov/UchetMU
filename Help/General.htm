Общая информация по работе с программой Учет МедУслуг

Программа Учет МедУслуг работает с единой базой данных. Файл базы данных имеет наименование UchetMU.mdb. Путь к файлу базы данных можно задать (или просмотреть) через пункт меню Сервис->Файл базы данных.

Ввод данных в программе Учет МедУслуг осуществляется с помощью заполнения полей таблиц. Используются 3 вида табличных полей:

текстовое поле  

поле с раскрывающимся списком

поле для ввода даты. При редактировании данного поля появляется элемент управления календарь.

Для выбора месяца нажмите кнопки в левом верхнем или правом верхнем углу календаря или щелкните мышью по наименованию месяца и выберите месяц. Щелчок по цифре года позволяет выбрать год.

Информация по некоторым справочникам и реестрам отображается через форму с двумя таблицами. Пользователь имеет возможность поменять для некоторых реестров связь между записями в верхней и нижней таблице. Для этого необходимо выделить записи в верхней и нижней таблицах. После этого выберите пункт меню Сервис->Связать с другим... и выберите новую запись для привязки в верхней секции.
