Формирование отчетов в программе Учет МедУслуг

Программное обеспечение Учет МедУслуг обеспечивает формирование следующих отчетов в формате Microsoft Excel.

Прейскурант цен. Для формирования данного отчета выберите пункт меню Отчеты->Прейскурант цен. Далее необходимо выбрать дату, на которую формируется прейскурант. Прейскурант цен содержит список услуг, имеющих калькуляцию на выбранную дату.
Реестр услуг. Для формирования данного отчета выберите пункт меню Отчеты->Реестр услуг. Далее необходимо выбрать период, за который будет сформирован реестр.
